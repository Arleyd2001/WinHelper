Comandos esenciales de Powershell

s

Script Soporte

Explora un completo menú interactivo que reúne las funciones más útiles para administrar, optimizar y solucionar problemas en sistemas Windows. Desde comprobaciones de conectividad y configuraciones de seguridad hasta la eliminación de archivos temporales, gestión de servicios y recuperación avanzada del sistema, este conjunto de comandos y submenús ha sido diseñado para facilitar las tareas de mantenimiento con opciones claras e intuitivas.

Comandos utilizados

Conectividad

- Test-Connection.
- ipconfig /all.
- ipconfig /release.
- ipconfig /renew.
- tracert.

Seguridad

- Get-Process
- Set-NetFirewallProfile

Eliminación de Archivos Temporales

- Remove-Item -Path "$env:temp\*".
- takeown /F [ruta].
- icacls [ruta].
- Remove-Item -Path "C:\Windows\Prefetch\*".
- Clear-DnsClientCache..

Gestión de Servicios

- Get-Service.
- Stop-Service -Name [servicio].
- Start-Service -Name [servicio].

Configuraciones del Sistema

- ncpa.cpl.
- appwiz.cpl.
- powercfg.cpl.
- sysdm.cpl.
- timedate.cpl.
- inetcpl.cpl.
- main.cpl.
- mmsys.cpl.
- desk.cpl.

Recuperación de Windows

- sfc /scannow.
- bootrec /fixmbr.
- bootrec /fixboot.
- rstrui.exe.
- DISM /Online /Cleanup-Image /RestoreHealth.
